# DNS查询过程

1. 查询浏览器DNS缓存是否命中，如果命中返回ip地址，查询结束
2. 查询操作系统(本地hosts文件)缓存是否命中，如果命中返回ip地址，查询结束
3. 操作系统将域名发送给本地域名服务器（运营商），本地域名服务器采用递归查询的方式分别向根域名服务器、顶级域名服务器、权威域名服务器查询自己的DNS缓存，如果命中返回ip地址，查询结束
4. 本地域名服务器代替客户端，采用迭代查询的方式发起请求
    - 向根域名服务器发起请求，返回顶级域名服务器地址`(.com)`
    - 向顶级域名服务器发起请求，返回权威域名服务器地址`(72lsy.com)`
    - 向权威域名服务器地址发起请求，最终得到ip地址
5. 本地域名服务器得到IP地址返回给操作系统，并把ip地址缓存
6. 操作系统得到IP地址返回给浏览器，并把IP地址缓存
7. 浏览器得到IP地址，并把IP地址缓存
